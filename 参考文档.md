## 第二阶段：数据类型与运算符

**目标：掌握python所有基本数据类型和运算操作，能灵活处理各种数据**

#### 基本数据类型

#### 字符串操作

切片：s[1:5]、s[::-1]（反转）

格式化：f-string f"我叫{name}，今年{age}岁"、.format()

常用方法：split、join、strip、replace、find、startwith、endwith、upper、lower

字符串是不可变类型，s.replace("a", "b")不会修改原字符串，而是返回新字符串

#### 容器类型

列表list：有序、可变、可重复，[1, 2, 3]，增删改查、排序、切片

元组tuple：有序、不可变、可重复，(1, 2, 3)，适合存不需要修改的数据

字典dict：键值对、无序（3.7+有序）、键唯一，{"name":  "张三"， "age":  “18”}

集合set：无序、不重复，{1, 2, 3}，适合**去重**和集合运算

类型转换与类型判断

类型转换：int()、str()、float()、list()、tuple()、set()、dict()

类型判断：type(x)查看类型、isinstance(x, int) 判断是否是某类型

#### 运算符

算术运算符：+、-、*、/（真除）、//（整除）、%、**（幂）

比较运算符：==（值相等）、!=、>、<、>=、<=

逻辑运算符：and、or、not

赋值运算符：=、+=、-=、*=、/=等

成员运算符：in、not in（判断元素是否在容器中）

身份运算符：is、is not（判断是否是同一个对象）

位运算符：&、|、^、~、<<、>>

三元表达式：x = a if 条件 else b

## 第三阶段：流程控制

**目标：掌握条件判断和循环，能用代码表达逻辑**

#### 条件判断

if / elif / else

嵌套条件判断

条件表达式（三元表达式）：result = "成年" if age >= 18 else "未成年"

#### 循环

for循环: 遍历列表、字典、字符串、range()

while循环：条件为真就一直执行

break：跳出整个循环

continue：跳过本次循环，进入下一次

pass：占位符，什么都不干

range()函数：range(10)、range（1，10）、range（0， 10， 2）

循环嵌套

match-case（Python 3.10+）：类似其他语言的switch-case

**一句话总结：for用来遍历，while用来重复**

#### 推导式

列表推导式：[x * 2 for x in range(10) if x > 3]

字典推导式：{k: v for k, v in items}

集合推导式：{x for x in list1}

生成器表达式：(x * 2 for x in range(10)) 惰性求值，省内存
